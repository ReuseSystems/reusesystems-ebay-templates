# Template Version History

## Version 1.0.0 (February 2025)

**Initial Release**

### Features
- ✅ Aircraft parts template with export controls
- ✅ Standard electronics template
- ✅ Responsive design (mobile/tablet/desktop)
- ✅ GitHub Pages hosting ready
- ✅ jsDelivr CDN support
- ✅ WP-Lister Pro compatible (Jinja2 templates)
- ✅ Bootstrap 4.6.2 framework
- ✅ Font Awesome 5.11.2 icons
- ✅ Pure CSS image gallery (no JavaScript)
- ✅ Tabbed content sections
- ✅ RAF aviation heritage branding
- ✅ Professional documentation

### Templates
1. **aircraft-parts.html**
   - Aircraft conditions warning
   - Export control compliance
   - Airworthiness disclaimers
   - 4 tabs (About, Terms, Shipping, Returns)

2. **standard.html**
   - General electronics template
   - Standard terms
   - 3 tabs (About, Shipping, Returns)

### CSS Files
- **style.css** (main stylesheet, 12KB)
- **responsive.css** (mobile/tablet styles, 4KB)

### Assets Required
- Logo: `images/logo.jpg` (user provided)
- External: Bootstrap, Font Awesome (CDN)

### Documentation
- README.md (comprehensive guide)
- QUICKSTART.md (5-minute setup)
- DEPLOYMENT-CHECKLIST.md (go-live checklist)
- IMAGES-README.md (asset guidelines)

### Scripts
- setup.sh (automated URL replacement)
- init-git.sh (repository initialization)

## Version Numbering

**Format:** MAJOR.MINOR.PATCH

- **MAJOR:** Breaking changes (e.g., 2.0.0)
- **MINOR:** New features, backwards compatible (e.g., 1.1.0)
- **PATCH:** Bug fixes, minor updates (e.g., 1.0.1)
